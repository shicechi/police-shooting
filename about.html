Hello,

This is an interactive map of the police shootings data that was crowdsourced from Deadspin in the fall of 2014.

This interactive map displays shooting incidents as markers on the map, since many of the incidents occurred in the same city and are overlapping, clustering of the markers is enabled. Clicking on a mark will provide a basic summary of the incident and a link to the story.

Resources used in creating this map:

  • Leaflet.js for constructing the map
  • OpenStreetMap for map data
  • Leaflet's Marker cluster plugin for clustering/decluttering the data points.
  • And of course jQuery and Bootstrap for the foundation.
